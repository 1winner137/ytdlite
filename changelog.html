Changelog

Release notes and upcoming features for yt-dlite

Version 1.0.0 April 30, 2025

Features

  • Core video and audio downloading functionality
  • Support for YouTube and other 1200 sites, playlists, and Shorts
  • Support for multiple quality options (video/audio)
  • Automatic video-to-MP3 conversion
  • Customizable output directory
  • Thumbnail extraction option
  • Cross-platform: Windows, Linux, macOS
  • Graphical user interface (GUI) and command-line interface (CLI)
  • Keyboard shortcuts for common actions
  • Detailed logging and better error messages
  • Lightweight executable with optimized performance
  • Conversion to different format both for audio and video

Upcoming Version

  • Individual download bar's or indicator per download in the GUI
  • Search Autosuggestion
  • Support cookies, for verified(sites require logins) downloads in Home-tab and Professional mode
  • Expanded language support (multi-language UI)
  • Further GUI enhancements for better usability
  • Bug fixes based on user feedback
  • More customization options (themes, settings)
  • Community-driven feature requests to be included
