Getting Started

Basic Usage (GUI)

  1. Launch YT-DLite from your applications menu.
  2. Paste a URL or enter search keywords in the search bar.
  3. Select your preferred format and quality.
  4. Choose destination folder and click Download.

Basic Usage (CLI)

To download a video via Command Line:

yt-dlite4cmd.exe [options] [urls]

Positional arguments:

  • [urls] - URL or Link to download

Options

  •     --video - any video URL to download
  •     --audio - any audio URL to download
  •     --format - Specify format for video or audio (e.g., mp4, webm, mp3)
  •     --output - Specify custom output directory
  •     --list-formats - List available formats for the given YouTube URL
  •     --resume - Resume partially downloaded files
  •     --help - Show this help message
